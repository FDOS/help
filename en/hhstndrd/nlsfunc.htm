nlsfunc

Adds NLS (national language support) functionality. This command is not available in FreeDOS yet. 
   nlsfunc [[d:][path]file]

Options

  d:path\file
  The full path to a file containing NLS information.
  If no file is given, nlsfunc will select its own default.
