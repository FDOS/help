fasthelp, fdhelp

Fasthelp clone for FreeDOS. Displays help on topic, Linux man-pages style. 
    fasthelp [topic]
Type 
    fasthelp
to see the list of topics.

Notes

Use the HELPPATH and LANG environment variables to specify the location and language of the help files respectively. The default path is ../help and default language is EN (english).

There is also the PAGER environment variable, which can be used to set an alternative program for displaying text help files. Calls to fasthelp will divert text files to the PAGER program.

See Set for more information on setting environment variables.

An identical command is fdhelp
