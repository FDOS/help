prompt

Changes the way the command prompt appears. 
   prompt [string]
Use path without specifying a string, to return it to its default appearance.

Special Character Codes

The following character sequences can be used to customise the command prompt: 
  $$   $
  $_   line break
  $B   |
  $D   the date
  $E   ASCII Escape character
  $G   >
  $H   Backspace
  $L   <
  $N   the current drive
  $P   the current drive and directory
  $Q   = 
  $T   the time
  $V   the name or version of the command shell

Examples

   prompt $p$g
This is the same as the default setting.
