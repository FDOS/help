Command: dosshell / fdshell

  DOSSHELL / FDSHELL is a clone of the MS DOSSHELL.

Syntax:

  DOSSHELL /?

Options:

  /?  Shows the help.

Comments:

  DOSSHELL opens a graphic user interface that is a clone of the MS
  DOSSHELL. If you know how to use Windows Explorer or other modern
  file management tools on Linux, you will understand this program
  immediately. You can navigate through the partitions that exist on
  your system with an arrow down symbol (below "help"), copy, move,
  delete, rename files, change attributes and create or delete
  directories by ALT-F and navigating to the position of your choice.
  It is also possible to minimize / maximize the size of the screen.
  Supports mouse, but only left mouse button.

Examples:

  Press ALT-F or ALT-O or ALT-V or ALT-H to come to the actions
  or click with the mouse on "File", "Options", " View" or "Help" and
  then navigate through the menu. Each menu position has a white
  character which you can choose with arrow up / arrow down or
  ALT-white character or mouse. Press SHIFT-F9 to come into command
  line. Simply test it!
