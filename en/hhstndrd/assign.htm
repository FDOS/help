assign


      assign [options] [d:=d:]
      assign /status
Reassigns disk operations from one drive to another. For example, to reassign disk operations for drive A: to drive B: 
  assign B:=A:
or
  assign B=A

Options

  /status
  Displays the current assignments.

  /M
  Attempts to use Upper Memory, instead of conventional memory,
  to store the ASSIGN.

  /U
  Uninstalls a previous ASSIGN command

  /shadow
  Makes ASSIGN's presence in memory difficult to detect.
  Subsequent uses of ASSIGN will issue a warning if the /shadow
  option isn't used (but still performs the command).

Notes

If no drives are named, and /status or /u are not used, then assign is reinstalled (clearing all previous drive assignments).
