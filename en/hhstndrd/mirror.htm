mirror

Mirrors information about the disk, such as the boot sector, file allocation table and root directory. 
  mirror d:  [/PARTN]
The information is stored at the end of the disk. This can be recovered if necessary using unformat.

Options

  d:
  The drive that you want to be able to recover later using unformat.


  /PARTN
  Backs up partition table information to a:\partnsave.fil
