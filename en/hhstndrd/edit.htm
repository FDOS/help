edit

Starts the DOS editor. 
   edit [[d:][path]file] [options]

Options

  [[d:]path]file]
  Specifies the file(s) to be opened. Wildcards * and ? can be used.

  /B
  Use a black-and-white (mono) display.

  /H
  Display using the highest video/text resolution available.

More Information

Edit has a comprehensive help system built in.

Limitations

Edit cannot currently open files greater than 64k in size.
