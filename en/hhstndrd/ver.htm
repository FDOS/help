ver

Displays FreeDOS version information. By default, only the version of the command shell (FreeCOM) is reported. 
   ver [options]

Options

   /R
   Shows the Kernel and DOS version numbers.

   /W
   FreeCOM Warranty.

   /D
   FreeCOM distribution information.

   /C
   FreeCOM acknolwedgements section. Lists contributors.
