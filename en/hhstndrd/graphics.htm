graphics

Allow the Prt Sc (Print Screen) key to print graphics screens. 
   graphics type [options]

Printer types

You can use the following for type: 
   HPPCL
   PCL
   EPSON              Works with many impact/ribbon printers
   ESC/P
   POSTSCRIPT
   PS
   LASERJET
   LASERJETII
   QUIETJET
   QUIETJETPLU
   HPDEFAULT
   DESKJET
   RUGGEDWRITER
   RUGGEDWRITERWIDE
Some of these are equivelent, such as POSTSCRIPT and PS.

The following types are not yet supported: 

   IBM PC color ribbon printer (color1, color4, color8)
   IBM convertible thermal printer
   IBM graphics printer, proprinter and quietwriter.

Options

   /B
   Use CGA background color

   /R
   Reverse colors: use black as white and white as black.

   /LCD
   /PRINTBOX:LCD
   /PRINTBOX:STD
   Ignored (not yet supported).
   These will allow you to set a screen width-to-height ratio other than
   4:3, which is currently always assumed.
