more

Displays a text file or the output of a command one screen at a time. 
  more < [d:][path]file
  command | more

Options

  d:path\file
  A text file that you want to display.

  command
  A command whose output you will pipe to the more program.

Examples

1. 
   mode < c:\config.sys
2. 
   fdisk /dump | more
