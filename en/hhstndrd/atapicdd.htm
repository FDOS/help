atapicdd.sys

Generic driver for IDE CD-ROM (ATAPI). Still under development.

   device=[drive][path]atapicdd.sys [options]

Options


   /D:name
   Sets the device name to name
   Default value: FDCD0000

   /N:x
   Sets the number of devices to look for to x.
   Default value: 1
