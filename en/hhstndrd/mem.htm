mem

Displays the amount of memory installed, and the amount available. 
   mem [options]

Options:

  /d
  Lists device drivers in memory

  /f
  Full information on memory blocks

  /u
  Display the programs that are loaded in memory.

  /e
  Full information on expanded memory

  /x
  Full information on extended memory
