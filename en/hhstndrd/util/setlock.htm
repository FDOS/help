Command: setlock

  SETLOCK is a utility to set and reset the lock keys on AT style
  keyboards

Syntax:

  setlock CL=[ON/OFF] NL=[ON/OFF] SL=[ON/OFF]
  Type SETLOCK without params to get help screen.

Options:

  CL = CAPSLOCK
  NL = NUMLOCK
  SL = SCROLL LOCK

Comments:

  - none -
