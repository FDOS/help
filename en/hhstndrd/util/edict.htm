Command: edict

  EDICT is an Enhanced Disk Image Creation Tool.
  It can create images of problem diskettes.

Syntax:

  edict options

Options:

  /d <drive>      Specify drive letter
  /f <filename>   Specify file name
  /p <limit>      Limit maximum number of additional passes
  /t <kind>       Override bios and select settings for disk type
                  (0=360Kib, 1=1.2MiB, 2=720KiB, 3=1.44Mib, a=auto)
  /c <number>     Override number of cylinders/tracks (0-1023)
  /r <number>     Override number of sides/heads per track (0-255)
  /s <number>     Override number of sectors per side (0-63)
  /b <number>     Override number of bytes per sector (1-8192)
  /h              Displays help information

Comments:

  Edict can also be used to create images of regular diskettes, e.g. for
  virtual machines.

Examples:

  Creates an image of diskette drive A: to C:\baddisk.img, overrides
  bios with setting 3 for 1.44Mib diskette and runs a maximum of
  10 passes.
    edict /d A: /f C:\baddisk.img /t 3 /p 10
