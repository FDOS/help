Command: fdbanner

  FDBANNER displays a FreeDOS banner in text mode.

Syntax:

  - none -

Options:

  - none -

Comments:

  You can add FDBANNER at the end of AUTOEXEC.BAT / FDAUTO.BAT
  to show that FreeDOS has booted.

Examples:

  In AUTOEXEC.BAT / FDAUTO.BAT:
    load this command
    load that command
    ...
    ...
  fdbanner  (at end of the file)

See also:

  autoexec.bat/fdauto.bat
