Command: cdrom / cdrom2

  CDROM is a toolkit for handling CD-ROM-drives.

  CDROM2 is a toolkit for handling CD-ROM-drives
  with additional audio support.

Syntax:

  CDROM / CDROM2 [EJECT|CLOSE|LOCK|UNLOCK|RESET] X:
  CDROM [drive:]
  CDROM2 [PAUSE|CONT|INFO|PLAYnn] X:
  CDROM2 [drive:]

Options:

  CD options:
  drive  the drive letter of the CD / Audio drive
  eject
  close
  lock
  unlock
  reset

  Audio options:
  pause  pause
  cont   continue
  info
  playnn  (xx is audio track number)

Comments:

  CDROM N:    Asks *CDEX 2.0+ if a CD-ROM exists.
  Errorlevels:
    0 okay,
    1 failed / no such drive.
  Shows a list of valid drives if invalid selection.
  Not all drives do all commands.

Examples:

  CDROM EJECT N:
  CDROM N:        Asks *CDEX 2.0+ if a CD-ROM exists
