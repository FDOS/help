Command: memteste

  MEMTESTE is a program for testing DRAM (main memory) of PC-compatible
  computers built on Intel 386 or higher processors.

Syntax:

  - none -

Options:

  - none -

Comments:

  Run MEMTESTE without drivers.
  MEMTESTE crashed on virtualbox but worked on a real machine and vmware.
  For more information see:
  https://gitlab.com/FreeDOS/util/memteste/-/blob/master/SOURCE/
  MEMTESTE/MEMTESTE.ASM?ref_type=heads

Examples:

  memteste
