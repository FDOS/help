emm386

The Expanded Memory Manager. This needs to be loaded in the config.sys 
   DEVICE=drive:path\emm386.exe [options]

Options

   NOEMS
   Do not enable expanded memory

   NOVCPI
   Do not use VCPI (must be used with NOEMS

   FRAME=hhhh
   Set the location of the page frame.
   Default is address E000.

   EMM=size
   Set the amount of expanded memory to create, in Kilobytes.
   The default is 8MB.

   VDS
   Experimental
   Enables Virtual DMA Services

   X=hhhh:hhhh
   Set a memory range for memory mapped devices
   E.g. X=D000-D800

   I=hhhh:hhhh
   Use memory that is reserved for some devices
   E.g:
       I=A000-AFFF
       I=B000-B7FF

   /VERBOSE
   detailed output when emm386 is loaded.
