graphics-ps, graphpin, graph-hp

You can use graphics instead. 
   graph-ps [options]
   graphpin [options]
   graph-hp [options]
Use graph-ps for PostScript-compatible printers.
Use graphpin for Epson-compatible dot-matrix printers.
Use graph-hp for HP-comatible PCL printers.

Options

  /I
  Prints the image colours reversed

  /B
  Prints the background in color, if possible

  /R
  Random dithering. Cannot be used with graph-ps

  /C
  Compatibility mode (reduced resolution)

  /E
  Economy mode (ink saving)

  /2
  use LPT2 (default is LPT1)

  /3
  use LPT3 (default is LPT1)
