country

Sets the international behavior of DOS. 
   country=nnn[,[mmm][,[d:][path]file]]

Options

  nnn
  The country code, as shown in the table below.

  mmm
  The code page, as shown in the table below.
  If not specified, a code page will be chosen automatically.

  d:path\file
  The pathname to a country-information file.
  If not specified, the file 'COUNTRY.SYS' is used instead.
