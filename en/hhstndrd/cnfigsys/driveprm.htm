driveparm

Defines parameters for block devices. 
   driveparm=[options]

Options

  /d:nnn
  The physical drive number, in the range 0-255. 0=A:

  /c
  Use this if the drive can detect when the door is open/closed.

  /f:n
  The drive type, corresponding to:
    0=160/180k or 320/360k
    1=1.2M
    2=720k (the default)
    3=[not defined]
    4=[not defined]
    5=hard disk
    6=tape
    7=1.44M
    8=R/W optical
    9=2.88M

  /h:nn
  The number of heads, in the range 1-99.

  /i
  Use this if the disk is a 3.5-inch drive

  /n
  Use this if the disk is a non-removable disk

  /s:nn
  The number of sectors, in the range 1-99

  /t:nn
  The number of tracks per side
