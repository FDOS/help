version

A config.sys command. Tells FreeDOS what version of DOS to report. 
   version=x.y
Where x and y are numbers (they each can have one or more digits).

The default value is 5.0

Notes

This command does not change the FreeCOM version number as reported by running 

   ver
However, the DOS version number is changed, as reported in: 
   ver /R

Example

  version=6.2
