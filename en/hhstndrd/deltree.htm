deltree

Delete an entire directory tree, including subdirectories. 
   deltree [options] [d:]path
Where d:\path is the directory tree to be deleted. Note that several directory trees can be deleted: 
   deltree [options] [d:]path1 [d:]path2 ...

Options

N.B. To use these options, they must come before the first path. Otherwise, they will be ignored. 
  /y
  Deletes without asking for confirmation.

  /v
  Reports the number of files and subdirectories deleted, and their
  total size.
