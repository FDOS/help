del, erase

Deletes files. 
   del [d:][path]file [options]
   erase [d:][path]file [options]

Options:

  d:path\file
  A file to delete.  A period may be used to specify all files in the
  current directory, and is the same as *.*

  /p
  Prompt before deleting the file.
